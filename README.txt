ACORN BCPNP Tool — GitHub Pages Deploy Package

Files:
- index.html (the app)
- (optional) acorn_bc_favicon_pro.ico / .png

Deploy (GitHub Pages):
1) Create a new GitHub repo (Public), e.g. acorn-bcpnp
2) Upload ALL files in this folder to the repo root.
3) Repo Settings → Pages → Source: Deploy from a branch
   Branch: main  Folder: / (root) → Save
4) Wait 30-60s. Your link will be:
   https://<your-username>.github.io/<repo-name>/

Tip:
- For iPhone: open link in Safari → Share → Add to Home Screen
